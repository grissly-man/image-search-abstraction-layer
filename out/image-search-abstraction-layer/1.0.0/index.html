image-search-abstraction-layer 1.0.0

  • User Story: I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  • User Story: I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  • User Story: I can get a list of the most recently submitted search strings.

API Usage

/add/{url}

Scrapes a (Pinterest Board) {url} and adds pictures to database.

NOTE: This method will only be useful with Pinterest Board URLs. Returns nothing. Only intended for building DB for demo purposes.

/search/{term}

Accepts a search parameter {term} and returns a weighted list of results

Does not yet implement pagination. Stay tuned!

For example:

https://image-meta-logan.herokuapp.com/search/cat
